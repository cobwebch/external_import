===============
External Import
===============

:Extension key:
   external_import

:Package name:
   cobweb/external_import

:Version:
   |release|

:Language:
   en

:Author:
   François Suter (Idéative), `typo3@ideative.ch <typo3@ideative.ch>`_

:License:
   This document is published under the `Open Publication License <http://www.opencontent.org/openpub/>`_.

:Rendered:
   |today|


----

Tool for importing data from external sources into the TYPO3 database, using an extended TCA syntax.
Provides a BE module, a Scheduler task, a command-line interface, reactions and an API.
